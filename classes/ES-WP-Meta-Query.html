Returns a simplified version of the meta-queries given as an argument.

protected queries_types_all_get(array<string|int, mixed> $meta_clauses) : array<string|int, mixed>

The simplification pertains to returning only the key/type values of each part of the meta-query. Also, the simplification takes care to flatten out the result.

If the meta-query is composed of multiple joining queries, these are processed by recursively walking through them, and calling this function to process each.
